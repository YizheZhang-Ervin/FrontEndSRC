# vue framework

## basic commands
```
vue create xxPorject # vue2
npm create vue@latest # vue3

npm install
npm run dev
npm run build
npm run test:unit
npm run lint

npm i --save ant-design-vue@4.x

# 修改源码
npx patch-package xxPackage

# 组件传值
- props和自定义事件
- 发布订阅：pubsub-js
- 管理状态容器：vuex 或 pinia 
- 全局事件总线：mitt
- 提供/注入：provide和inject
- 双向数据绑定：v-model
- 插槽：slot
- 引用(获取实例/DOM)：ref与$parent
- 获取组件的属性与事件：useAttrs
```
