# WebSSH

1. 前端【webssh-vue】
- 依赖
  - npm init vue
  - npm install xterm
  - npm instakk xterm-addon-fit
- 运行
  - cd webssh-vue
  - npm install
  - npm run dev 或 npm run build

2. 前端【webssh-vue-cdn】
- 直接访问index.html

2. 后端【webssh-ws】
- 依赖
  - npm install utf8
  - npm install ssh2
  - npm install ws
- 运行
  - cd webssh-ws
  - npm install
  - node app.js
